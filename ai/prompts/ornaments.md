# UI Ornament Prompts

Manual ChatGPT prompt batch for small UI ornaments. Save outputs into
`GPT-ASSETS/ui/ornaments/`. Ten ornaments total.

## Base style

Small painterly ornament tile, 128x128 PNG, early modern Spanish
tercio era, transparent or flat magenta background, no text, no
modern, no fantasy, no glow. Designed to layer over panels and
dividers.

## Negative style

No text. No glow. No drop shadow. No busy background. Keep the
stroke weight consistent across the set so they read as a family.

## Output contract

- 128x128 PNG, transparent background preferred.
- Subject fills about 60 percent of the canvas.
- Save with magenta background only when the cleanup script must
  remove it. For ornaments prefer **transparent** from the start.

## Per-ornament prompts

1. `ui_corner_gold_tl.png` — Top-left gold filigree corner, warm
   gold, no text.
2. `ui_corner_gold_tr.png` — Top-right gold filigree corner, mirror
   of the top-left, warm gold, no text.
3. `ui_corner_gold_bl.png` — Bottom-left gold filigree corner,
   mirror, warm gold, no text.
4. `ui_corner_gold_br.png` — Bottom-right gold filigree corner,
   mirror, warm gold, no text.
5. `ui_divider_gold_short.png` — Short horizontal gold divider,
   about 96 px wide, warm gold, no text.
6. `ui_divider_gold_long.png` — Long horizontal gold divider,
   about 192 px wide, warm gold, no text.
7. `ui_rivet_iron.png` — Square iron rivet, weathered, no text.
8. `ui_rivet_gold.png` — Square gold rivet, polished, no text.
9. `ui_wax_seal_blood.png` — Cracked red wax seal, simple
   cross stamp, no text.
10. `ui_banner_small_burgundy_cross.png` — Small hanging
    burgundy cross banner, tattered edge, no text.

## Mature SFW variant

Wax seal uses a non-gore dark red. The banner is worn, not
dripping. No variant required for this set.
