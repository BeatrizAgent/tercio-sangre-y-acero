# Model Notes

First pack is SDXL-only by default. Do not add Pony, Illustrious, SD1.5, Flux, anime, NSFW, celebrity, POI, fetish, or modern military models unless explicitly approved.

Use model files only after:

1. Exact source URL is recorded.
2. Exact model version ID is recorded.
3. License terms are copied into `license_review.md`.
4. `commercial_use_reviewed` is intentionally set.
5. `enabled` is set to true for one reviewed entry at a time.
