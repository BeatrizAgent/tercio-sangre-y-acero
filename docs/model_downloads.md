# Model Downloads

Use the example manifest as a shopping list, not an automatic download list. Every entry starts disabled.

## Starter Pack

- RealVisXL V5.0
- Juggernaut XL
- Morion Helmet
- Arquebus
- Medieval armour SDXL
- Cinematic Portrait SDXL
- Zavy's Old Portraits SDXL
- Old Style SDXL
- game_icons_painted
- Renaissance workshop

## Workflow Pairings

Portraits: RealVisXL + Morion Helmet + Cinematic Portrait + optional Zavy Old Portraits.

CGI/backgrounds: Juggernaut XL + Old Style + Renaissance workshop/style.

Inventory icons: SDXL checkpoint + game_icons_painted + Game Icon SDXL.

Equipment: Juggernaut XL or RealVisXL + Morion Helmet + Arquebus + Medieval armour.

UI mockups: Juggernaut XL + Renaissance Style + Old Style.

Sprites: use only as reference or concept; final sprites need manual cleanup.

## Fill Version IDs

For Civitai, open the model page, choose the exact reviewed version, and copy its model version ID into `civitai_model_version_id`. Do not guess from the page URL. If permissions are unclear, leave `enabled=false`.
